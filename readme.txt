** Este é o exercicio do curso Git**

**Criação do diretorio:
** comando: mkdir curso
**Criação do arquivo:
** comando: touch readme.txt
**Edição do conteúdo do arquivo
** comando: vim readme.txt
**Para o apontamento do diretorio no GitHub
** comando git remote add origin https://github.com/leoghibaudo/curso.git
** Criar o arquivo de configuação
** comando: vim .git/config (o ponto na frente do diretorio git é para ficar oculto)
**Após é necessário fazer o push no branch master
** comando: git push origin master
